# Sidebar Icons

To use custom icons in the admin sidebar, upload your icon files (SVG or PNG) here with these exact names:

- `dashboard.svg` (or `.png`) - Dashboard icon
- `services.svg` (or `.png`) - Services icon
- `portfolio.svg` (or `.png`) - Portfolio icon
- `packages.svg` (or `.png`) - Packages icon
- `bookings.svg` (or `.png`) - Bookings icon
- `messages.svg` (or `.png`) - Messages icon
- `about.svg` (or `.png`) - About Us icon
- `users.svg` (or `.png`) - Users icon
- `site.svg` (or `.png`) - View Site icon
- `logout.svg` (or `.png`) - Logout icon

**Icon Requirements:**
- Recommended size: 18x18px to 24x24px
- Format: SVG (preferred) or PNG with transparent background
- Color: Icons will be automatically colored white, so use black/simple designs
- The icons will be displayed at 18x18px in the sidebar

If an icon file is missing, the sidebar will display a fallback dash character (â€”).
